# Perhitungan-Hari-Kerja-dengan-NETWORKDAYS-Excel
Mini project menggunakan Microsoft Excel untuk menghitung jumlah hari kerja antara dua tanggal dengan fungsi NETWORKDAYS. Perhitungan mempertimbangkan tanggal merah dan cuti agar hasil lebih akurat. Project ini melatih pengelolaan waktu kerja secara efisien.
